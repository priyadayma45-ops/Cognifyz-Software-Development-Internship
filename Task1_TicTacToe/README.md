# 🎮 Tic Tac Toe Game

> **A simple, interactive, and fun console-based Tic Tac Toe game built with Java.**
> Developed as part of the **Cognifyz Technologies Software Development Internship**.

---

## 📌 Project Overview

This project is a classic **2-player Tic Tac Toe Game** that runs in the Java console.

Players take turns placing **X** and **O** on a 3×3 board. The game automatically checks for a winner, prevents invalid moves, and detects draw situations.

This project demonstrates the implementation of Java programming fundamentals through a real-world mini application.

---

## ✨ Features

✔️ Interactive Console-Based Gameplay

✔️ Two Player Mode (X vs O)

✔️ Beautiful 3×3 Game Board

✔️ Input Validation

✔️ Prevents Overwriting Existing Moves

✔️ Automatic Winner Detection

✔️ Draw Detection

✔️ Clean & Easy-to-Understand Code

✔️ Beginner Friendly Project
